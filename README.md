## Проектная работа «Закрывающий тег»

### Цель работы:
cоздать интерактивную страницу-рефлексию о пути обучения с адаптивным дизайном, сложными CSS-анимациями и эффектами, публикацией на GitHub Pages

### Стек технологий
- HTML5 (семантическая разметка, <dialog>, SVG)
- CSS3 (CSS-переменные, Grid, Flexbox, анимации, фильтры)
- SVG (символы, inline-вставка, анимация)
- JavaScript (готовый функционал для лайков и диалогов)
- Адаптивный дизайн (Mobile First, резиновая вёрстка)
- GitHub Pages (публикация проекта)

### Ключевые особенности
- Сложные градиенты — создание сетки с плавным переходом
- Резиновая вёрстка — использование clamp() для всех размеров
- SVG-анимация — управление 4 элементами с разными таймингами
- CSS-фильтры — создание уникальных эффектов для изображений
- Режимы смешивания — mix-blend-mode, difference
- Адаптивный <dialog> — корректное отображение на мобильных
- Интерактивные элементы с кастомными эффектами
